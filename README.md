Під час навчання я опанував наступні методи та структури:
• DFS та BFS: Поглибив навички застосування пошуку в глибину та ширину для дерев і складних графів.
• Tries (Префіксні дерева): Ця структура була для мене новою. Я навчився використовувати її для ефективного збереження та пошуку слів.
• Priority Queue (Heaps): Опанував роботу з купами для реалізації черг із пріоритетами, що є критичним для оптимізації багатьох алгоритмів.
